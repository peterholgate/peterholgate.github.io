Using This Site


Use the LINKS on this site to get to any of the listed sites in 2 clicks (plus scroll on the mobile version)


On the PC version of this site, links can be accessed via the alphabetic TOP BAR and SIDE BAR


On the MOBILE version of this site, links can be accessed via the alphabetic SIDE BAR only


Links on this Website usually go to UK WEB SITES (click on browser left arrow to return), you will need to sign in to your accounts


PC PHOTOS are viewed via Google Photos; click on the 3 vertical dots (vertical ellipsis) to see a slideshow


PC users can return to this Website by closing the current browser tab


Mobile PHOTOS are viewed by selecting a start photo, then clicking on the 3 vertical dots to see the slideshow option


Android users return to this Website using the back arrow eg the navigation button < at the bottom right


IPhone users return to this Website using the < symbol at the top left


Use the outlined LEFT and RIGHT arrows to navigate to adjacent pages on this site


The outlined UP arrow takes you to the top of the current page or screen


Clicking on the [PFH] Icon takes you to the Home Page

 

Windows PC Security


If you are ‘infected’ with Toolbars, Search Bars or other Adware or


Malware (Viruses, Trojans etc), try the following


(All software is available via [Page 10 Utility] of this website)


     Run Glary Utilities


     Click on [1 Click Maintenance]


Click on [Scan for Issues]


Select [Automatically Repair]


Run MalwareBytes


     Select Drive C: via [File Explorer]


     Click on [This PC]


     Right Mouse Click on [Local Drive C:]

Select [Scan with Malwarebytes


Prevent Re-infection


     Run Glary Utilities (1 Click) every week


     Run Malwarebytes default (Scan) every week


     Always have ONE Anti Virus and ONE Firewall active

    

     Ensure that your Anti-Virus is up to date


     Ensure that your Operating System and Apps/Programs etc are


updated regularly (Glary Utilities can help with this)
