Windows PC Security


  • If you are ‘infected’ with Toolbars, Search Bars or other

Adware or Malware, try the following

  • All software is available on Page 10 (Utility) of this

            website


Run Ccleaner


  • Analyse [Windows] and clean
  • Analyse [Applications] and clean
  • Takes a few Minutes


Run MalwareBytes Anti-Malware


  • Full Scan on all Drives
  • Follow the on screen guide
  • RESTART (ie RE-BOOT)
  • Can Take Hours!


To Prevent Re-infection


  • Run CCleaner daily
  • Run Glary Utilities (1 Click) weekly
  • Run MalwareBytes (Threat Scan) every 2 weeks
  • Always have ONE Anti Virus and ONE Firewall

            running but NOT more than one of either

  • Ensure that your Anti-Virus is up to date
  • Ensure that your Operating System and

            Apps/Programs etc are updated regularly

Useful Terms


What is a BROWSER or a WEB BROWSER?


  • A web browser, or browser, is an application used to

access and view websites. Common web browsers include                    Microsoft Edge, Google Chrome, Mozilla Firefox, and

Apple Safari.


What is the difference between a PROGRAM and an APP?


  • The only difference is a technical one. While both are the same, an 'application' is a computer program launched and dependent upon an operating system to execute. ‘App’ is

short for Application.


What is a COMPUTER OS or OPERATING SYSTEM?


  • The three most common operating systems for personal computers are Microsoft Windows, Mac OS X, and Linux. Basically a piece of software that enables programs etc to

use your computer hardware.
